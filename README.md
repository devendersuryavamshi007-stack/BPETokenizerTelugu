# BPE Tokenizer for Telugu Language

## 📊 Results

- Vocabulary Size: 6289
- Compression Ratio: 5.38

## 📚 Dataset
Telugu Wikipedia text (translated and collected manually)

## 🧠 Approach

- Built Byte Pair Encoding (BPE) tokenizer from scratch
- Implemented:
  - Character-level tokenization
  - Pair frequency counting
  - Iterative merging
  - Encoding and decoding pipeline

## ⚙️ How to Run

1. Open the notebook
2. Upload dataset (text file)
3. Run all cells

## 🚀 Features

- Supports Telugu Unicode text
- Efficient compression of repeated patterns
- Fully custom tokenizer implementation
